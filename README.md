## GitHub Pages Setup

1. Upload these files to your GitHub repository.
2. Go to **Settings**.
3. Click **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the **main** branch and the **/root** folder.
6. Save the settings.

Your site will then be published through GitHub Pages.
